# System Information Commands

## 📌 Overview

Used to gather information about system and user.

## 🧠 Commands

- whoami → current logged-in user
- id → user ID and group ID
- hostname → system name
- uname → system details
- env → environment variables

## ⚙️ Examples

whoami
id
hostname
uname -a (uname alone dont do much but when used with -a or -r do more )
(havent learned much about them , to know more about them use the help commands on linux like man or --help)
env

## 💡 Explanation

These commands help identify the system, OS, and user environment.

## 🔥 Use Case

Used in cybersecurity for reconnaissance after gaining access.

## 📝 Summary

Helps understand the system you are working on.
