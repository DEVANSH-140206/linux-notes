# Navigation Commands

## 📌 Overview

Commands used to move around directories in Linux.

## 🧠 Commands

- pwd → shows current working directory (tells where u are currently)
- ls → lists files and folders present currently where u are
- cd → change directory (move to one place to other)
- cd .. → move one step back

## ⚙️ Examples

pwd
ls
cd Documents (will move you to location "documents" or you can say will change your directory to documents)
cd ..

## 💡 Explanation

Linux uses a hierarchical file system starting from root (/).
Navigation commands help move through directories.

## 🔥 Use

Used in every Linux task, including hacking environments.

## 📝 Summary

Navigation is the foundation of working in Linux.
