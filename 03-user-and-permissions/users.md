# Users in Linux

## 📌 Overview

Linux is a multi-user system where multiple users can exist.

## 🧠 Commands

- whoami → current user
- id → user and group ID
- who → all the current logged in users

## ⚙️ Examples

whoami
id
who

## 💡 Explanation

Each user has permissions and access levels.

## 🔥 Use Case

Used to check privileges during penetration testing.

## 📝 Summary

Understanding users is important for access control.
