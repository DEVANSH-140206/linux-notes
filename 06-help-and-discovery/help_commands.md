# Help Commands

## 📌 Overview

Used to learn about other commands.

## 🧠 Commands

- man → manual pages
- --help → quick help
- apropos → search commands
- which → find command location

## ⚙️ Examples

man ls
ls --help
apropos network (when u dont know what to do with a particular command)
which python

## 💡 Explanation

Linux provides built-in documentation for commands.

## 🔥 Use Case

Used when learning new tools or commands.

## 📝 Summary

Help commands make learning Linux easier.
