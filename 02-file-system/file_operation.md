# File System Basics

## 📌 Overview

Linux file system is hierarchical and starts from root (/).

## 🧠 Concepts

- Everything is treated as a file
- Files are organized in directories
- Root directory is "/"

## 💡 Explanation

Linux organizes data in a structured way for easy navigation and control.

## 🔥 Use Case

Understanding file system is essential for navigating and exploiting systems.

## 📝 Summary

File system knowledge helps in managing and locating files efficiently.
