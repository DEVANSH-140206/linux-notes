# Network Commands

## 📌 Overview

Used to check network configuration and connections.

## 🧠 Commands

- ip → shows IP configuration
- ifconfig → network interface info
- netstat → network connections
- ss → socket statistics

## ⚙️ Examples (havent gone through yet)

ip a
ifconfig
netstat -tuln
ss -tuln

## 💡 Explanation

These commands show how system communicates over network.

## 🔥 Use Case

Used in scanning, enumeration, and network troubleshooting.

## 📝 Summary

Networking commands help understand system connectivity.
